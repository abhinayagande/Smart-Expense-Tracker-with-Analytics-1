# Smart Expense Tracker with Analytics

## Project Overview
Smart Expense Tracker with Analytics is a Python-based application that helps users track their expenses, analyze spending habits, and visualize financial trends through interactive charts and dashboards.

## Features
- View and analyze expense data
- Category-wise spending analysis
- Total expense calculation
- Monthly expense trends
- Pie chart visualization
- Bar chart visualization
- Identification of highest spending category
- Interactive dashboard using Streamlit

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- NumPy
